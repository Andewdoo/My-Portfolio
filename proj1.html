Smart Timetable Calendar

A smart scheduling tool integrating an AI engine (in progress) with a fully built frontend, allowing users to create and manage events intuitively.

About this project

This project tackles the lack of smart AI-powered scheduling tools by building a timetable app designed for anyone managing multiple deadlines. It offers a clean frontend with event creation, calendar views, and early support for sharing schedules with friends. The goal is to integrate AI to automatically plan tasks, optimize timelines, and reduce the stress of manual scheduling, while future backend storage integration will enable seamless collaboration. Beyond solving a real planning problem, the project is also a hands-on learning experience in building full-stack applications as a student.

Key features

  • Feature A — Clean and intuitive UI.
  • Feature B — AI automatic scheduling, AI can directly talk to the backend and manipulate events.
  • Feature C — Share timetables and schedules, all timetables are saved to backend data storage.

Languages

React · Javascript · CSS · MongoDB
